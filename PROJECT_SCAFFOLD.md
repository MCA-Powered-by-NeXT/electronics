# Basic Electronics Technician Course Scaffold

This repository is organized as a workforce-aligned static course website for Basic Electronics Technician training. The structure is HTML/CSS only so it can be hosted on GitHub Pages or copied into any static site workflow.

The project includes a 6-week sequence, 12 core lessons, a Week 1 Tinkercad supplemental lesson, 8 required labs, weekly quizzes, practical checkoffs, final project materials, and instructor support pages.

Safety boundary: all student lab content is limited to low-voltage battery-powered DC training circuits. Student pages warn against wall outlets, mains voltage, open power supplies, vehicle electrical systems, charged capacitors, unknown energized equipment, and unsafe DMM current measurements.
