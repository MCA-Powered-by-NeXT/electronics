# Basic Electronics Technician Course Scaffold

This test branch converts the original day-based electronics content into a larger workforce-aligned static course website. The starter structure is intentionally HTML/CSS only so it can be hosted on GitHub Pages or copied into any static site workflow.

The project follows the current implementation brief: 6 weeks, 2 core lessons per week, 8 required labs, weekly quizzes, practical checkoffs, final project materials, and instructor support pages.

Safety boundary: all student lab content is limited to low-voltage battery-powered DC training circuits. Student pages warn against wall outlets, mains voltage, open power supplies, unknown energized equipment, and unsafe DMM current measurements.
