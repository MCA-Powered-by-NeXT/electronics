Daily pacing notes for a 6-week everyday Basic Electronics Technician class. Plan 2-3 hours per class day.

This guide assumes five class meetings per week. Each class block should include opening review, explanation, guided practice, student work, documentation, and a short exit check.

Make-Up and Retry Guidance

  • Students who miss a daily class should complete the linked day page and any required lab or quiz item.
  • Safety-critical checkoffs must be retried after coaching if unsafe behavior occurs.
  • Students who miss the Tinkercad day should complete a guided simulation or instructor-observed demonstration before Week 2 circuits.
  • Document practical retries so completion decisions are transparent.
