Instructor Only

This guide assumes five class meetings per week. Each class block should include opening review, textbook-style explanation, guided demonstration, student practice, documentation, and a short exit check. Use the student textbook reader when students need more conceptual explanation before or after the activity.

Course Delivery Pattern

Documentation / quiz / exit check20-40 minCollect notes, repair tickets, checkoffs, quizzes, or exit tickets.

Make-Up and Retry Guidance

  • Students who miss a daily class should complete the day page, the matching Textbook Reader section, and the matching daily quiz.
  • Hands-on labs require instructor-observed make-up or a comparable demonstration of the same competency.
  • Safety-critical checkoffs must be retried after coaching if unsafe behavior occurs.
  • Quiz corrections are appropriate for learning, but practical safety errors require demonstration-based retry.
  • Document practical retries so completion decisions are transparent.
