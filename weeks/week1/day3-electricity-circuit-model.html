Student Reading: Learn This First

A circuit needs a source, path, load, and return. Say that sentence first, then point to each part in a drawing, simulation, or physical trainer.

The source provides electrical energy for the circuit. In class, the source is an approved low-voltage battery or trainer. Do not use unknown sources or wall outlets.

The path is made by conductors such as wires, breadboard rows, terminals, or traces. The path must actually connect the intended points. Color, neatness, or closeness does not prove connection.

The load is the part doing useful work, such as an LED making light or a buzzer making sound. A load needs a complete path through it and back to the source.

The return completes the loop back to the source. Beginners often forget the return. Current does not travel to the LED and disappear. Missing return means the intended path is open.

Path conditions describe what is happening. A closed circuit has the intended complete path. An open circuit has a break. A short circuit has an unintended low-resistance path that may bypass the load or protection.

Write this explanation: “In this circuit, the source is __, the path goes through __, the load is __, and the return is __. If __ opens, the circuit will __.”

Lesson Work

  1. Draw or label a simple source-path-load-return circuit.
  2. Identify one closed path, one open path, and one short or unintended path example.
  3. Trace the path aloud from source to return.
  4. Explain what happens if the switch opens, the LED is reversed, or the return wire is missing.
