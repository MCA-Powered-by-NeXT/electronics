Week 6 Day 4: Final Demonstration and Rubric Scoring

2-3 hour class block for project demonstration, explanation, documentation review, and rubric scoring.

Week 6Previous DayNext Day

Day 4 Purpose

Students demonstrate their final project and explain the safety habits, circuit path, measurements, troubleshooting steps, and documentation used to complete it. The demonstration is both a technical check and a communication check.

Learning Targets

  • Demonstrate safe final project operation.
  • Explain the project using correct vocabulary.
  • Show measurements or test evidence.
  • Respond to instructor questions professionally.
  • Use documentation to support the demonstration.

Student Demonstration Script

  1. State the project purpose.
  2. Identify the source, path, load, and return.
  3. Describe one safety check used before power-up.
  4. Show the project function or final test result.
  5. Explain one measurement, fault, or correction.
  6. State what the documentation proves.

Instructor Follow-Up Questions

  • What would happen if the return path opened?
  • Where is current limited in your project?
  • What test gave you the best evidence?
  • What did you change after the first build?
  • What would you improve if you built it again?
