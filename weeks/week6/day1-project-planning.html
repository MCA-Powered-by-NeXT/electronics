Student Reading: Learn This First

Define the project function before choosing parts. Say what the project will do in one sentence.

Identify the source. The source must be approved for the class. Write the type of source and why it is acceptable.

Identify the control. The control starts, stops, or changes the output. It may be a switch, pushbutton, or another approved classroom control.

Identify the load or output. The load might be an LED, buzzer, lamp, or other approved low-voltage output. Write what the output will do.

Identify protection or current limiting. If the project uses an LED, the resistor must be part of the LED path. If a fuse or connector is used, explain its purpose.

Define proof of success. Write how you will know the project works: visual output, sound, measurement, continuity check, or demonstration statement.

Lesson Work

  1. Select an approved project option.
  2. Write function, source, control, output, return, and protection/current-limiting notes.
  3. Sketch or describe the circuit path.
  4. Get instructor approval before building.
