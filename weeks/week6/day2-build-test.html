Week 6 Day 2: Build and Initial Test

2-3 hour class block for building the approved final project and recording initial measurements.

Week 6Previous DayNext Day

Day 2 Purpose

Students build the approved project using safe work habits and record evidence as they go. The instructor checks that power is controlled, parts match the plan, and the build can be inspected before first power-up.

Learning Targets

  • Build according to an approved plan.
  • Inspect before applying power.
  • Record build steps and measurements.
  • Perform an initial function test safely.
  • Identify whether the project is ready for final troubleshooting or needs revision.

Build Checklist

  • [ ] Parts match approved plan.
  • [ ] Source polarity identified.
  • [ ] Current-limiting components installed where needed.
  • [ ] Source, path, load, and return can be traced.
  • [ ] No loose wires or visible shorts.
  • [ ] Wiring is clean enough to inspect.
  • [ ] Instructor approval before first power-up if required.

Common Build Mistakes

  • Building before the plan is approved.
  • Forgetting the return path.
  • Reversing LED or source polarity.
  • Leaving out a current-limiting resistor.
  • Changing several things without updating the documentation.

Exit Ticket

Describe the current state of your project: working, partially working, or not working yet. Include one piece of evidence.
