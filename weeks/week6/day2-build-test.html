Student Reading: Learn This First

Build in sections. Do not wire everything at once. Build source, control, protection, load, and return in an organized sequence.

After each section, stop and inspect. Check for correct row, secure wire, proper polarity, current-limiting placement, and no exposed conductor.

Before power, compare the build to the plan. If the build does not match the approved plan, stop and ask before continuing.

Initial testing should be calm and documented. The project may work, partly work, not work, or be unsafe to test. Each result is useful if recorded honestly.

If the project does not work, do not rebuild randomly. Write the symptom and choose the first safe check.

Write a build note: what was built, what was inspected, what happened during first test, and what the next step is.

Lesson Work

  1. Build the project in approved stages.
  2. Inspect each stage before continuing.
  3. Complete pre-power inspection.
  4. Run first test and record result.
  5. Write next safe action if the project does not work.
