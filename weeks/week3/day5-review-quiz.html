Student Reading: Learn This First

Week 3 is about turning the DMM into a safe evidence tool. You learned that a meter reading is useful only when the task, setup, circuit state, endpoints, units, and meaning are clear.

Start your review with the question. If you need to know whether an approved source has voltage, use DC voltage setup on the powered approved source. If you need to know a resistor value, use resistance with power off. If you need to know whether a path is complete, use continuity with power off and named endpoints.

Do not mix up powered and unpowered tests. Voltage is the powered test in this beginner course. Resistance and continuity are power-off tests. Current mode is a stop-and-ask topic unless the instructor controls the setup.

Documentation turns a reading into evidence. “8.9” is not evidence by itself. “Battery 1 measured 8.9 V DC across source terminals” is evidence. The difference is context.

Continuity is especially useful for Week 4 because schematics are path maps. If the schematic says two points should connect, continuity can help test an unpowered physical path. But only if you know the endpoints.

Fun fact: troubleshooting often improves when students stop asking “What is broken?” and start asking “What question am I trying to answer?” The second question leads to the correct test. The first question often leads to poking things and hoping.

Before the quiz, complete this sentence: “The safest meter user is not the fastest person; it is the person who knows the __, chooses the __, confirms the __, and records the __.”

Beginner Extra Help: One Question, One Setup

The fastest way to review Week 3 is to pair each question with one correct setup. “Is voltage present?” means DC voltage setup on an approved powered source. “What is this resistor value?” means resistance setup with power off. “Is this path complete?” means continuity setup with power off and named endpoints.

Do not let the meter setting lead the thinking. Let the question lead the meter setting. The meter is the helper, not the boss. If the meter were the boss, it would probably schedule a meeting called “Why Are My Leads in the Wrong Jack?” and nobody wants that meeting.

Day 5 Purpose

Students confirm they understand meter setup, safe measurement habits, continuity testing, and current-mode awareness before using these skills with components and schematics. The review emphasizes choosing the correct test for the question being asked. Students should leave Week 3 able to say: "This is the question, this is the correct meter setup, this is the circuit state, and this is what the reading means."

Learning Targets

  • Choose the correct DMM setup for voltage, resistance, or continuity scenarios.
  • Explain powered vs unpowered measurement requirements.
  • Recognize current-mode hazards and stop rules.
  • Write measurement notes with units and meaning.
  • Correct common DMM misconceptions.
  • Connect DMM and continuity skills to Week 4 component and schematic work.

  • DMM parts: display, dial, jacks, leads, probes.
  • DMM setup routine: task, circuit state, jacks, function, endpoints, units.
  • DC voltage measurement on approved low-voltage classroom sources.
  • Resistance measurement with power off and assigned isolated components.
  • Continuity testing of wires, switches, fuses, and simple unpowered paths.
  • Current-mode awareness and stop rules.
  • Documentation with number, unit, item tested, circuit state, and meaning.

Week 3 Quiz

  1. What should you check before using a DMM?
  2. Which jack does the black lead normally use?
  3. Which jack should the red lead use for voltage, resistance, and continuity practice?
  4. Why is current mode treated with extra caution?
  5. When is it safe to measure resistance?
  6. What does continuity mean?
  7. How should a closed switch test when using the correct terminals?
  8. Why are units important in measurement records?
  9. What should you do if a meter setup looks wrong?
  10. Explain the difference between voltage measurement and continuity testing.
  11. Write a safe measurement note for a 9 V battery reading.
  12. Write a safe continuity note for a wire tested with power off.

Correction Prompts

  • Explain the difference between voltage, resistance, and continuity tests.
  • Write the safe setup for measuring an assigned battery/source.
  • Write the safe setup for checking a fuse.
  • Explain why power-off testing matters for resistance and continuity.
  • Rewrite one measurement note so it includes units, item, state, and meaning.
  • Write one current-mode stop rule from memory.

Week 4 Readiness Checklist

  • [ ] I can inspect and set up a DMM before measuring.
  • [ ] I can choose voltage, resistance, or continuity based on the question.
  • [ ] I can record measurements with units and meaning.
  • [ ] I can test continuity only when power is off.
  • [ ] I can name the endpoints in a continuity test.
  • [ ] I can explain why schematic tracing and continuity testing go together.
  • [ ] I can describe a circuit path before testing it.

Exit Ticket
