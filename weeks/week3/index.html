Module Description

This is a core workforce skill week. Students learn that meter safety is not optional: the dial, jack, range, circuit state, and probe placement must match the measurement being attempted.

DMM current warning: Never place a meter set to current mode directly across a battery or voltage source. Current measurements require the correct jack, current range, internal fuse protection, and instructor approval.

Learning Outcomes

  • Identify DMM parts, jacks, leads, and common symbols.
  • Select correct settings for DC voltage, resistance, and continuity.
  • Record measurements with units and notes.
  • Perform continuity tests only on circuits verified power-off.
