Student Reading: Learn This First

Continuity means a complete conductive path exists between two named points. The key words are complete, conductive, and named points.

Continuity is a power-off test. Disconnect or verify power is off before checking wires, switches, fuses, connectors, or path sections.

Name endpoints clearly. Do not write “wire good.” Write “continuity from switch output to resistor input is present with power off.” The endpoints give the result meaning.

Switches change continuity. A switch open should interrupt the path. A switch closed should complete the path across the correct terminals. Record switch position every time.

Fuses require interpretation. An open fuse is not just a failed part. It is evidence that the path opened, possibly because of a fault. Record it and ask why.

Use this note template: endpoints tested, circuit state, meter result, and meaning.

Lesson Work

  1. Test continuity of known-good and open wires with power off.
  2. Test switch terminals in open and closed positions.
  3. Test fuse samples and interpret open/closed results.
  4. Trace one unpowered trainer path using named endpoints.
  5. Write complete continuity notes.
