Week 4 Day 1: Components, Wiring, Polarity, and Workmanship

2-3 hour class block introducing common electronic components, physical part recognition, polarity, careful handling, and wiring habits that support troubleshooting.

Day 1 Purpose

Students identify common parts and describe how each part appears in beginner circuits. The focus is recognizing the part, knowing its purpose, knowing whether polarity matters, and handling parts carefully. Students also learn that wiring is communication: another technician should be able to follow the path without guessing.

The uploaded beginner electronics PDF uses simple component drawings and practical examples to introduce parts such as resistors, capacitors, diodes, LEDs, switches, wires, and fuses. This lesson adapts that beginner-friendly approach into hands-on identification stations using classroom-safe parts and low-voltage trainers.

Safety boundary: Students handle disconnected components, parts trays, and assigned low-voltage training circuits only. Capacitors are discussed as components, but this course avoids stored-energy hazards and does not use charged capacitors for student handling.

Learning Targets

  • Identify resistors, LEDs, diodes, capacitors, switches, fuses, wires, connectors, terminals, buzzers, and lamps used in beginner circuits.
  • State a basic purpose for each component in technician language.
  • Identify which parts are polarity-sensitive or orientation-sensitive.
  • Inspect component condition and leads before building.
  • Use careful wiring habits that make paths easier to trace.
  • Write component notes that include name, purpose, polarity/handling note, and possible fault.

Wiring Habits That Help Future Troubleshooting

  • Keep source and return visually consistent when possible, but do not let color replace tracing.
  • Use short, direct jumpers when possible so the path is visible.
  • Avoid long loops that hide parts or cross over polarity marks.
  • Do not force component leads into a breadboard.
  • Keep exposed metal from touching unintended points.
  • Place resistors so they are actually in the current path they are supposed to protect.
  • Route wires so another student can trace source, control, load, and return.
  • Disconnect power before changing any wire or component.

Technician Note Practice

Weak NoteBetter Note
"Small part with stripes.""Resistor with color bands; used to limit current in LED path. Value should be checked before use."
"Light part.""LED output. Polarity matters; use flat side/lead length to orient and include current-limiting resistor."
"Switch thing.""Pushbutton switch opens/closes the control path. Terminals must be identified before wiring."

Common Mistakes

  • Calling every small part a resistor.
  • Ignoring LED or diode polarity.
  • Letting wire color replace actual path tracing.
  • Building a circuit that works once but cannot be inspected.
  • Forgetting that switch terminals matter.
  • Using the word "bad" without describing visible evidence.

Knowledge Check and Exit Ticket

1. Which components in today's tray may be polarized?
2. Why does an LED need a resistor?
3. What does a switch do in a circuit path?
4. Why should wiring be neat and traceable?
5. What is one visible sign that a wire or connector may cause a fault?
Exit ticket: Name three components, each component's purpose, and one handling note.
