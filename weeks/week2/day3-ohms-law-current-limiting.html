Day 3 Purpose

Students use Ohm's Law as a simple technician tool. The focus is practical: voltage, current, and resistance are related, so changing one value changes what the circuit may do.

Learning Targets

  • Use V = I x R for simple calculations.
  • Identify which quantity is being asked for in a word problem.
  • Explain current limiting in a simple LED circuit.
  • Show work with units.
  • Use calculations to support, not replace, safe inspection.

Main Concept: Ohm's Law

V = I x R
Voltage = Current x Resistance

Ohm's Law helps technicians predict what may happen in a circuit. In this course, calculations stay simple and support safe low-voltage decisions.

Classroom rule: Write the formula, substitute numbers with units, solve, then write the answer with units.

Problem-Solving Routine

  1. Read the problem and underline the known values.
  2. Identify what the problem is asking for: V, I, or R.
  3. Write the formula.
  4. Substitute the known numbers.
  5. Solve carefully.
  6. Write the answer with the correct unit.
  7. Ask whether the answer makes sense for a beginner low-voltage circuit.

Current Limiting

An LED can be damaged if too much current flows through it. A resistor placed in the intended path limits current and protects the LED. The resistor is not optional; it is part of the safe circuit design.

Battery + -> Resistor -> LED -> Battery -
If resistance is too low, current may be too high.
If resistance is too high, the LED may be very dim or off.

Common Mistakes

  • Using the wrong unit: Ohms, volts, and amps are not interchangeable.
  • Skipping the setup: Guessing at a formula leads to wrong answers.
  • Forgetting that current can be small: Beginner LED currents are often written in milliamps.
  • Thinking math makes a circuit safe: Inspection and correct wiring still come before power.

Student Explanation Prompt

Explain to a new student why a current-limiting resistor protects an LED. Use the words voltage, current, resistance, and path.
