Student Reading: Learn This First

Describe the circuit before building it. The approved source provides voltage. The resistor limits current. The LED produces light. The return completes the loop.

Inspect before power. The source must be approved. The resistor must be in the LED path. The LED polarity must be correct. The return must be connected. No jumper should bypass the resistor or load.

Build in sequence. Place the source connection. Place the resistor path. Place the LED with correct orientation. Place the return. Inspect the whole loop before applying power.

When the circuit works, do not stop at “it lights.” Explain why it lights: the path is complete, the LED is oriented correctly, and current is limited by the resistor.

If the LED does not light, follow the loop. Check source, path, resistor, LED polarity, and return. Make one correction at a time.

Write a lab note with source used, resistor value, LED polarity note, expected operation, actual operation, and any correction made.

Lesson Work

  1. Draw the LED circuit before building.
  2. Build the circuit using the assigned source, resistor, LED, and wires.
  3. Inspect the circuit before applying power.
  4. Test and record actual operation.
  5. If needed, troubleshoot one change at a time and verify.
