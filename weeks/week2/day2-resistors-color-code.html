Day 2 Purpose

Students learn why resistors are used and how technicians identify resistor values before placing parts into a circuit. The day emphasizes slow, accurate reading and clear documentation instead of guessing from memory.

Learning Targets

  • Explain what a resistor does in a simple circuit.
  • Read a 4-band resistor color code at an introductory level.
  • Recognize tolerance as an allowed range, not a mistake.
  • Select an appropriate resistor before building a current-limited LED circuit.
  • Document resistor value with units.

Why Resistors Matter

A resistor controls current by adding resistance to the path. In a simple LED circuit, the resistor protects the LED by limiting current. Without the resistor, the LED may draw too much current and fail.

Battery + -> Resistor -> LED -> Battery -
The resistor is in series with the LED so current must pass through it.

Technician Tips

  • Separate similar-looking colors under good lighting.
  • Use the tolerance band to help identify the reading direction.
  • Record the unit with the number.
  • Do not guess resistor values when building an LED circuit.
  • Keep decoded resistors in labeled groups during lab setup.

Common Mistakes

  • Reading from the wrong end: Look for spacing and tolerance band placement.
  • Forgetting the multiplier: The first two bands are not the full value.
  • Leaving off ohms: 330 and 330 Ω are not equally clear technician notes.
  • Assuming tolerance means the resistor is bad: A real resistor may be slightly above or below the marked value.
