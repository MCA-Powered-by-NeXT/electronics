Week 2 Day 2: Resistors and Color Code

2-3 hour class block focused on resistor purpose, value markings, and color-code practice.

Day 2 Purpose

Students learn why resistors are used and how technicians identify resistor values before placing parts into a circuit.

Learning Targets

  • Explain what a resistor does in a simple circuit.
  • Read a 4-band resistor color code at an introductory level.
  • Recognize tolerance as an allowed range, not a mistake.
  • Select a resistor before building a current-limited LED circuit.

4-Band Resistor Model

Band 1 = first digit
Band 2 = second digit
Band 3 = multiplier
Band 4 = tolerance

Students should practice slowly and record values with units. The goal is accuracy, not speed.

Technician Tips

  • Separate similar-looking colors under good lighting.
  • Record the unit with the number.
  • Use a DMM later to verify resistance when required.
  • Do not guess resistor values when building an LED circuit.

Knowledge Check

1. What does a resistor do?
2. What does the multiplier band do?
3. Why does tolerance matter?
4. Why should technicians verify part values before building?
Exit ticket: Decode one resistor and write its value with units.
