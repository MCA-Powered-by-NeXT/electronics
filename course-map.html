Course Map

Six-week, everyday class sequence for Basic Electronics Technician. Each week is organized into five 2-3 hour daily class meetings and aligned to the README course outline.

Daily Class Length

Plan each day for a 2-3 hour block. A typical daily block includes a short opening review, concept instruction, guided practice, student work time, documentation, and an exit ticket or signoff.
