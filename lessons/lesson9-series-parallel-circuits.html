Lesson 9: Series and Parallel Circuits

Estimated time: 2-3 hours. Learn how current paths change circuit behavior and fault symptoms.

Previous LessonWeek 4Next LessonRelated LabWeek 4 Quiz

Learning Outcomes

  • Build and identify series and parallel circuits.
  • Explain current path behavior in series and parallel branches.
  • Predict what happens when one load opens in each circuit type.
  • Use voltage checks to locate simple open faults.

Key Vocabulary

Series: one path for current. Parallel: multiple current paths. Branch: one parallel path. Voltage drop: voltage used across a component. Open fault: a break that stops current in a path.

Main Concept

In a series circuit, all current must pass through every part. One open can stop the whole circuit. In a parallel circuit, each branch has its own path, so one open branch may leave other branches working. Parallel branches can increase total current draw, so the source and wiring must be appropriate.

Series: + -> R1 -> LED1 -> LED2 -> - Parallel: + -> R1 -> LED1 -> - + -> R2 -> LED2 -> -
Rule of thumb: In series, resistance adds. In parallel, branch currents add.

Technician Tips

  • If all loads are dead, check the source and shared return first.
  • If one parallel branch is dead, compare it to a working branch.
  • Use continuity only with power off; use voltage measurements when checking powered low-voltage training circuits.
  • Give each parallel LED its own current-limiting resistor.

Common Mistakes

  • Using one resistor for several parallel LEDs and expecting equal brightness.
  • Assuming more resistors always means more total resistance; that is true in series, not always in parallel.
  • Removing one series part and wondering why the whole loop is off.

Knowledge Check

1. What happens to two series LEDs if one LED is removed?
2. Why does adding a parallel branch make the source work harder?
3. Where would you measure voltage first if all loads are off?
