# Site Map

This branch reorganizes the electronics repository into a static 6-week Basic Electronics Technician course website.

## Top-level pages
- index.html: course dashboard and start page
- main.html: compatibility landing page for older links
- syllabus.html: full student-facing syllabus
- course-map.html: week, lesson, lab, quiz, and assessment map
- competencies.html: certificate competencies and checkoff overview
- tools-equipment.html: required tools, parts, consumables, and safety equipment

## Course folders
- weeks/: weekly module landing pages
- lessons/: 12 core lesson pages
- labs/: 8 required lab worksheets
- quizzes/: weekly quiz pages
- assessments/: practical checkoffs and rubrics
- instructor/: instructor-only support pages
- assets/css/styles.css: shared responsive and print-friendly stylesheet

## Legacy migration
Original day1.html through day10.html remain in place while content is migrated into the larger 6-week structure.
