Lab 5: Component Identification and Schematics

Week 4 required lab. Match physical parts to schematic symbols and build from a simple diagram.

Week 4Lesson 7Lesson 8Week 4 QuizContinuity Checkoff

Safety Warning

Power stays off during identification: Component identification, schematic tracing, and continuity checks are performed on disconnected parts and low-voltage training boards only.

Purpose and Skills Practiced

  • Identify common components by appearance and function.
  • Match components to schematic symbols.
  • Trace a simple source-control-load-return path.
  • Build a simple circuit from a schematic card.

Tools and Parts

  • Mixed component tray
  • Resistors, LEDs, switches, fuses, wires, connectors, capacitors, and diodes
  • Schematic symbol cards
  • Breadboard or training board
  • DMM for power-off continuity verification

Setup Diagram

Schematic card example:
BT1 + -> F1 -> SW1 -> R1 -> LED1 -> BT1 -

Match:
BT1 = battery
F1 = fuse
SW1 = switch
R1 = resistor
LED1 = LED load

Procedure

  1. Sort the mixed tray into component types.
  2. Match each component to a schematic symbol card.
  3. Record one function for each component.
  4. Trace the schematic from source to return using your finger or pencil.
  5. Build the circuit with power disconnected.
  6. Use continuity mode only where directed to verify wiring paths.
  7. Ask the instructor to inspect the build before applying low-voltage battery power.

Reflection

Which schematic symbol was easiest to remember, and which needs more practice?

Cleanup and Reset

  1. Disconnect all power.
  2. Return components to labeled bins.
  3. Return symbol cards and schematic cards.
  4. Clear the board for the next team.
