Lab 7: Fault Board Troubleshooting

Week 5 required lab. Diagnose planted faults using inspection, measurements, and documentation.

Safety Warning

Training faults only: Fault boards must be low-voltage DC trainer boards. Disconnect power before changing wiring or using continuity mode.

Purpose and Skills Practiced

  • State a circuit symptom clearly.
  • Inspect for visible faults.
  • Use DMM measurements to isolate the fault.
  • Correct one fault at a time and verify the repair.
  • Complete a repair ticket.

Tools and Parts

  • Low-voltage fault board
  • DMM
  • Schematic or wiring card
  • Replacement jumper wires, resistors, fuses, and LEDs as directed
  • Repair ticket worksheet

Setup Diagram

Fault board stages:
Source -> protection -> control -> resistor/load -> return

Test order:
Visual -> source voltage -> return path -> control path -> load -> verify

Procedure

  1. Record the symptom before touching the board.
  2. Perform a visual inspection with power disconnected.
  3. Use the schematic or wiring card to identify source, control, load, and return.
  4. Apply low-voltage power only after instructor approval.
  5. Measure source voltage and document the result.
  6. Disconnect power before continuity checks or wiring changes.
  7. Correct one fault and retest.
  8. Complete the repair ticket and final verification.

Troubleshooting Questions

  1. Why should you record the symptom before changing anything?
  2. What does source voltage present but no load operation suggest?
  3. Why should only one correction be made before retesting?

Reflection

What measurement or observation gave you the best evidence?

Cleanup and Reset

  1. Return the board to the instructor for reset.
  2. Remove temporary wires only if directed.
  3. Turn off the DMM and power source.
  4. Submit the repair ticket.
