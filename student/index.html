How To Use This Course

Start each day from your week page. Each daily lesson includes the class purpose, learning targets, agenda, practice work, documentation prompts, and an exit ticket. Use the Textbook Reader for longer day-by-day explanations of the concepts behind the activities. Quiz packets are organized by week and include a 10-question quiz for each daily lesson.

Student lab boundary: Work only on assigned classroom training circuits and sources. Ask your instructor before changing wiring, changing a meter setting, or using any source not assigned for the activity.
